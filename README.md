# Tarea Día 2 - Ejercicios de C#

## 📋 Ejercicios

### 1. El Portero de Discoteca
Implementa la lógica del portero que evalúa el acceso según la edad.

### 2. Calculadora Humilde (Switch)
Crea una calculadora básica usando `switch` que suma, resta o multiplica.

### 3. Detector de Pares
Determina si un número es PAR o IMPAR usando el operador `%`.

## 🚀 Cómo correr el proyecto

### Ejecutar todos los tests
```bash
dotnet test
```

### Ejecutar con más detalle
```bash
dotnet test --verbosity detailed
```

### Restaurar dependencias
```bash
dotnet restore
```

## ✅ Verificar tu progreso

Los tests te dirán si tu código funciona correctamente. Al principio todos fallarán, pero a medida que completes los ejercicios, verás más tests pasando:

```
✅ Test Passed
❌ Test Failed
```

¡Completa el código en cada archivo `.cs` hasta que todos los tests pasen!
